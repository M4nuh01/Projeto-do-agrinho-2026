# Projeto-do-agrinho-2026
# 🌱 Raízes do Futuro

Um site interativo desenvolvido com HTML, CSS e JavaScript sobre sustentabilidade na agricultura, destacando o equilíbrio entre produção agrícola e preservação ambiental.

---

## 🌍 Sobre o projeto

O projeto **Raízes do Futuro** foi criado com o objetivo de apresentar de forma visual e interativa a importância da agricultura sustentável e do uso consciente dos recursos naturais.

Ele aborda temas como:
- Agronegócio moderno 🚜  
- Recursos naturais 🌿  
- Relação entre campo e natureza 🤝  
- Polinização e importância das abelhas 🐝  
- Impactos ambientais ⚠️  
- Agricultura consciente 🌱  
- Futuro sustentável ⚖️  

---

## ✨ Funcionalidades

- 🖥️ Capa inicial interativa com imagem de destaque    
- 🧭 Menu de navegação com indicação de seção ativa  
- 📱 Layout responsivo (funciona em celular e PC)  
- 🌿 Fundo visual temático com imagem da capa  
- 🎨 Design moderno com tipografia elegante  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
